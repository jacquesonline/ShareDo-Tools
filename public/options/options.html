Notifications are enabled from this time. Uses the server's local timezone.

Browser Context
Keep Browser Context Outside Hours
When enabled, the Playwright browser context remains alive and page checks continue recording metrics outside office hours. Only alert delivery is suppressed. When disabled, the browser context is shut down outside hours to save memory.
